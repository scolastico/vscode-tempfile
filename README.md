# vscode-tempfile

This extension immediately creates a temporary file and opens it in a new tab.

## Configuration

| Config               | Description | Default                                                       |
| -------------------- | ----------- | ------------------------------------------------------------- |
| tempfile.newFilePath |             | {{tmpdir}}/{{YYYY}}{{MM}}{{DD}}\_{{HH}}{{mm}}{{ss}}{{SSS}}.md |
